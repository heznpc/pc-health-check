PROBLEM

"수상한 항목 30건"이 떴을 때, 무엇이 진짜 위협인가?

한국 PC는 뱅킹·공공기관·쇼핑몰 보안 플러그인으로 가득합니다. 이것들은 전부 범용 스캐너에는 수상해 보이지만, 실제로는 정상입니다. 컴퓨터에 익숙하지 않은 사용자는 구분할 길이 없어 "전부 무시"하거나 "전부 삭제"하는 양극단을 택하게 됩니다.

8가지 진단 → 1개의 HTML 리포트.

PRIVACY · AUDITABLE

"개인정보 안전합니다"라는 말 대신, 코드로 증명.

모든 네트워크 호출은 소스 코드에 있습니다. VirusTotal 조회는 SHA-256 해시만 전송하고, 파일 본문은 컴퓨터를 떠나지 않습니다. 선택적 Sysinternals 다운로드도 별도 helper에 격리되어 있습니다.

해시만 전송, 파일 업로드 안 함

Get-FileHash로 로컬에서 SHA-256 계산 후 GET 요청. 알려지지 않은 해시면 "unknown" 반환, 자동 업로드 없음.

텔레메트리 0건

사용 통계·에러 리포트 어떤 것도 수집하지 않습니다. 외부 호출은 옵트인 VirusTotal 조회와 선택적 Sysinternals 다운로드뿐입니다.

로컬 캐시만

VT 응답은 %LOCALAPPDATA% (Windows) 또는 ~/Library/Caches (macOS)에 48시간 저장. 클라우드 계정 없음.

읽을 수 있는 소스

컴파일된 바이너리 0개. PowerShell · Python · Bash 약 3,700줄. grep -R "Invoke-RestMethod\|Invoke-WebRequest\|urlopen" scripts/로 모든 외부 호출 확인 가능.

scripts/vt-lookup.ps1 L142-L160 · GitHub 
# SHA-256 해시 계산 - 로컬에서만 수행
$hash = (Get-FileHash -Path $FilePath -Algorithm SHA256).Hash.ToLower()

# VirusTotal API v3 조회 - 해시만 전송, 파일 업로드 없음
$url     = "https://www.virustotal.com/api/v3/files/$hash"
$headers = @{
    'x-apikey' = $apiKey
    'accept'   = 'application/json'
}

try {
    $resp = Invoke-RestMethod -Uri $url -Headers $headers -Method Get
    return $resp.data.attributes.last_analysis_stats
} catch {
    if ($_.Exception.Response.StatusCode -eq 404) {
        return 'unknown'   # 알려지지 않은 해시 - 업로드 안 함
    }
}
