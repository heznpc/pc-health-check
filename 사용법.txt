======================================================
  PC 건강검진 - 사용법
======================================================

[ 이 프로그램은 무엇인가요? ]

내 컴퓨터가:
  - 바이러스/악성코드에 감염되었는지
  - 해커에게 원격 조종당하고 있는지
  - 몰래 비트코인 채굴기로 쓰이고 있는지
  - macOS가 System Data/Developer로 뭉뚱그린 저장공간의 정체가 무엇인지
를 검사해 신호등(🟢🟡🔴)으로 알려주는 도구입니다.

Windows와 macOS 모두 지원합니다.


[ Windows 사용법 ]

  1. "검사하기.bat" 파일을 더블클릭
  2. 메뉴에서 숫자 입력:
     [1] 빠른 검사 (1분)
     [2] 정밀 검사 (6분) - 5분 CPU 관찰 포함 (추천)
  3. 검사가 끝나면 브라우저에 결과가 뜸


[ macOS 사용법 ]

  SwiftUI 앱으로 실행:
  1. 공증된 DMG가 배포된 버전이면 앱을 Applications로 옮겨 바로 실행합니다.
     소스 ZIP 버전은 "Mac앱실행.command" 우클릭 → "열기"로 빌드합니다.
  2. 소스 모드는 매 실행 시 증분 빌드로 앱과 내장 진단 스크립트를 최신 상태로 맞춘 뒤 자동으로 엽니다.
  3. 앱에서 [지금 검사]를 누르면 요약 화면에 남은 공간과 직전 검사 이후 변화가 표시됩니다.
  4. [왜 변했나]에서 증가한 경로와 추적되지 않은 APFS/swap/임시 파일 변화를 구분해 확인합니다.
  5. [공간 정리]에서 항목을 선택하고 오른쪽 [정리 검토]를 누르면 미리보기가 열립니다.
  6. 대상 경로, 예상 크기, 종료해야 할 프로세스와 재다운로드 비용을 확인한 뒤 개별 승인하세요.
  7. 설치 앱은 번들 ID로 다시 확인한 뒤 앱과 정확히 귀속되는 데이터만 휴지통으로 이동합니다.
  8. Simulator는 항목을 선택한 뒤 오른쪽 [보존]/[보존 해제]로 관리합니다.
     Shutdown 상태이며 보존되지 않은 기기만 [삭제 검토]가 나타납니다.
  9. "매시간 확인"을 켜면 로컬 용량 표본을 제한된 개수만 기록하고 위험할 때 알림을 보냅니다.
  10. 툴바의 리포트 메뉴는 브라우저로 열거나 다른 사람에게 보여줄 때만 사용하세요.

  스크립트 메뉴로 실행:
  1. 터미널을 열거나, "검사하기.command" 더블클릭
     - 처음에는 "열기 권한이 없다"고 뜰 수 있어요.
     - 해결: 파일 우클릭 → "열기" 선택 (한 번만 하면 됨)
  2. 메뉴에서 숫자 입력:
     [1] 빠른 검사 (Mac Edition)
     [2] 이전 결과 다시 보기
     [3] 공유용 결과 열기
  3. 검사가 끝나면 일반 리포트와 공유용 리포트가 함께 생성됨


[ 리포트 보는 법 ]

  🟢 안전   = 정상입니다. 걱정 안 하셔도 돼요.
  🟡 확인   = 위험하진 않지만 한 번 확인해보세요.
  🔴 위험   = 즉시 조치가 필요합니다.

  각 항목 옆 [구글 검색], [VirusTotal] 버튼으로 바로 조사.

  macOS 공유용 리포트:
  - 검사결과_공유용.html
  - PC 이름, 사용자 이름, 홈 디렉터리 경로를 자동으로 가립니다.
  - 공유 전에는 내용을 한 번 더 확인하세요.


[ 로컬 전용 / 보안 원칙 ]

  - 기본 검사는 이 Mac 안에서만 실행됩니다.
  - AI 연동, OpenAI/Claude/Codex API 호출, 토큰 사용은 없습니다.
  - 검사 결과를 서버로 업로드하지 않습니다.
  - VirusTotal 조회를 직접 켠 경우에만 SHA-256 해시가 외부 조회됩니다.
  - SwiftUI 앱은 삭제를 자동 실행하지 않습니다.
  - 지원되는 캐시 정리는 고정된 로컬 레시피만 사용하며 미리보기와 개별 승인을 거칩니다.
  - 관련 Codex/Claude/Chrome/Xcode/Simulator 작업이 실행 중이면 정리를 차단합니다.
  - Codex 세션 JSONL, Claude 로컬 에이전트 작업공간, SDK, Simulator 런타임에는 정리 버튼이 생기지 않습니다.
  - 정리 후에는 다시 검사하며 영수증을 ~/Library/Application Support/PC Health Check/cleanup-receipts/ 에 남깁니다.
  - 승인한 정리가 진행 중일 때 앱 종료를 선택하면 안전한 거래 경계와 영수증 처리가 끝날 때까지 정상 종료를 기다립니다.
  - 앱 제거는 즉시 영구 삭제하지 않고 ~/.Trash/PC Health Check-... 폴더로 이동합니다.
  - 저장공간 감시는 자동 정리를 하지 않으며 20GB 미만 또는 직전보다 8GB 이상 줄 때만 알립니다.
  - 검사별 경로 크기 이력은 최대 180개, 시간별 여유 공간 표본은 최대 336개만 로컬에 보관합니다.
  - 이력 파일은 ~/Library/Application Support/PC Health Check/ 아래에 600 권한으로 저장되며 외부로 전송되지 않습니다.
  - SwiftUI 앱은 시스템 목록·폼·툴바를 사용하는 네이티브 유틸리티입니다.
  - macOS가 Mail/Messages/Safari/앱 컨테이너를 숨기면 Full Disk Access 안내가 표시됩니다.
  - SwiftUI 빠른 검사는 큰 Simulator/SDK 폴더에서 오래 멈추지 않도록 크기 측정 시간을 제한합니다.
    정확한 용량까지 다시 재고 싶으면 터미널에서 PCH_STORAGE_DU_TIMEOUT=0 bash scripts/scanner.sh 로 실행하세요.


[ VirusTotal 자동 조회 켜기 (선택) ]

  켜면: 70여개 백신 DB로 자동 2차 검증
  끄면: 기본 검사만 (빠름)

  켜는 방법:
  1. https://www.virustotal.com 회원가입 (무료)
  2. 우측 상단 프로필 아이콘 → "API Key" 복사
  3. 다음 중 하나를 선택:
     - 권장: VT_API_KEY 환경 변수로 현재 사용자에게만 설정
     - 스크립트 메뉴: data/config.example.json을 data/config.json으로 복사한 뒤 로컬 파일을 편집
     - SwiftUI 앱(소스 빌드 포함): ~/Library/Application Support/PC Health Check/config.json 편집
  4. 파일을 사용하는 경우:
     - "enabled": true
     - "apiKey": "YOUR_API_KEY"
  5. 저장 후 검사 재실행

  ※ 파일 내용은 절대 서버로 안 보내고 해시(지문)만 조회합니다.
  ※ 무료 한도: 분당 4건, 일일 500건. 캐시로 자동 관리.


[ 정밀 검사 팁 (Windows) ]

  정밀 검사는 "5분간 가만히 있기"가 핵심입니다.
  유튜브/게임을 모두 끄고 기다리세요.
  그래야 "아무것도 안 하는데 CPU를 쓰는 범인"을 잡습니다.


[ Sysinternals 도구 (Windows, 선택) ]

  기본값은 첫 실행 시 다운로드 여부를 확인합니다.
  data/config.json 에서:
    "sysinternals": { "autoDownload": true }
  로 바꾸면 Microsoft 공식 도구(sigcheck, autorunsc)를
  Microsoft 서명 검증 후 자동 다운로드하여 훨씬 상세한 자동실행 분석을 제공합니다.

  저장 위치: %LOCALAPPDATA%\PC건강검진\tools\


[ 결과가 "위험"으로 나왔다면? ]

  1. 당황하지 말고 리포트의 "위험" 항목을 읽으세요.
  2. 프로그램 이름을 구글에 검색
     - 유명 프로그램이면 오탐 가능성
     - 이상한 이름이면 악성코드일 확률 높음
  3. VirusTotal 버튼 클릭하여 상세 조회
  4. Windows: "바이러스 및 위협 방지" → "전체 검사"
     macOS: Malwarebytes for Mac 무료판 2차 검사
  5. 추가 도움이 필요하면:
     - Windows: Malwarebytes Free, Emsisoft Emergency Kit
     - macOS: Objective-See 도구 (KnockKnock, LuLu)


[ 폴더 구성 ]

  PC건강검진/
  ├─ 검사하기.bat         ← Windows 실행
  ├─ 검사하기.command     ← macOS 실행
  ├─ Mac앱실행.command    ← macOS SwiftUI 앱 실행
  ├─ 사용법.txt           ← 이 파일
  ├─ 검사결과.html         ← 스크립트 모드 검사 후 자동 생성
  ├─ 검사결과_공유용.html  ← macOS 스크립트 모드 공유용 결과
  ├─ data/
  │  ├─ whitelist.json    ← 한국 프로그램 DB
  │  ├─ explain.json      ← 쉬운 설명
  │  └─ config.example.json ← 비밀정보 없는 설정 예시
  └─ scripts/
     ├─ menu.ps1          ← Windows 메뉴
     ├─ scanner.ps1       ← Windows 스캐너
     ├─ scanner.sh        ← macOS 스캐너
     ├─ cleanup.sh        ← macOS 정리 미리보기/승인 실행
     ├─ storage_watch.sh  ← 저장공간 급감 감시
     ├─ schedule.sh       ← 감시 LaunchAgent 켜기/끄기
     ├─ scanner_helper.jxa.js ← macOS 데이터 집계
     ├─ build_macos_swift_app.sh ← macOS SwiftUI 앱 빌드
     ├─ monitor.ps1       ← 5분 CPU 관찰 (Windows)
     ├─ report.ps1        ← Windows HTML 생성
     ├─ report.jxa.js     ← macOS HTML 생성
     ├─ vt-lookup.ps1     ← VirusTotal 조회
     ├─ sigcheck-helper.ps1  ← Sysinternals sigcheck
     └─ autorunsc-helper.ps1 ← Sysinternals autorunsc

  실제 API 키 설정은 배포본에 포함되지 않습니다.
  - 스크립트 모드: 무시되는 data/config.json
  - SwiftUI 앱(소스 빌드 포함): ~/Library/Application Support/PC Health Check/config.json
  SwiftUI 앱의 검사 결과와 리포트는 위 Application Support 아래 results 폴더에,
  변경 기록과 감시 표본은 Application Support 아래 소유자 전용 상태 파일에 저장됩니다.


[ 자주 묻는 질문 ]

Q. 검사하기.bat 더블클릭해도 까만 창만 뜨고 끝나요.
A. 관리자 권한이 필요할 수 있습니다.
   우클릭 → "관리자 권한으로 실행"

Q. "이 스크립트는 실행할 수 없습니다" 오류
A. PowerShell 관리자로 열고:
   Set-ExecutionPolicy -Scope CurrentUser RemoteSigned

Q. macOS에서 "확인되지 않은 개발자" 경고
A. 파일 우클릭 → "열기" 선택 (한 번만)
   또는 시스템 환경설정 → 보안 및 개인정보 보호 → "확인 없이 열기"

Q. 다른 컴퓨터에도 쓸 수 있나요?
A. 네. PC건강검진 폴더 전체를 USB로 복사해 쓸 수 있습니다.
   설치 필요 없음.

Q. 어떤 백신을 같이 쓰면 좋나요?
A. 이 도구는 '진단 리포터'이지 백신이 아닙니다.
   백신은 별도로:
   - Windows: Windows Defender (기본) + Malwarebytes Free
   - macOS: macOS Gatekeeper (기본) + Malwarebytes for Mac
